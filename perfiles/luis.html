Conclusión sobre la especificidad en CSS

La especificidad en CSS determina qué estilo se aplica cuando varias reglas afectan al mismo elemento. Comprender cómo funcionan los selectores permite controlar mejor el diseño sin necesidad de usar demasiadas clases o IDs,lo que hace el código más limpio y fácil de mantener.

Selectores que me costaron entender

  • :nth-child() → Permite seleccionar elementos según su posición dentro de un contenedor.
  • ::before → Permite agregar contenido decorativo antes de un elemento.
  • Selector descendiente (ul li a) → Selecciona enlaces dentro de listas.

¿Qué es Mobile-First?

Mobile-First es una metodología de diseño web donde primero se crean los estilos para dispositivos móviles y luego se agregan adaptaciones para pantallas más grandes usando media queries. Esto permite crear sitios más rápidos, accesibles y mejor adaptados a los dispositivos actuales.
